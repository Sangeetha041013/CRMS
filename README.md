Campus Recruitment Management System (CRMS)

Description :
A web-based system that connects students and companies to streamline campus recruitment. It allows students to apply for jobs and companies to manage hiring efficiently.

Features :
* Admin dashboard & reports
* Company job posting & application management
* Student profile, job search & application tracking
  
Tech Stack :
* Frontend: HTML, JavaScript
* Backend: PHP
* Database: MySQL
* Server: XAMPP

Modules :
* Admin
* Company
* Student
  
Key Benefits :
* Reduces manual work
* Faster recruitment process
* Organized data management
  
Conclusion :
A simple and efficient system to manage campus recruitment digitally.
